# Dynamic Pricing Optimization System Using AI

This project implements an AI-driven dynamic pricing system that analyzes data and adjusts prices dynamically using machine learning.

## Project Structure

**backend/**
Contains the backend API and business logic.

**frontend/**
Contains the user interface and client-side functionality.

**model/**
Contains trained machine learning models used for prediction.
